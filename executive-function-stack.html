Executive Function
Scaffolding Stack

The ADHD O/S operates as a governed, closed-loop executive function support system — beginning not at the task list, but at the moment of friction itself. A Governance Layer sets the rules for how the system interprets and intervenes. Each layer below feeds the next. The system learns without losing its integrity.

Governance → Intelligence → Orchestration → Execution → Adaptation → ↺
Governance Layer Umbrella · Governs all layers

Barkley-informed epistemic discipline
and anti-shame guardrails.

As the system becomes more adaptive and prescriptive, a Governance Layer ensures it remains evidence-aligned, non-moralizing, and epistemically honest. It asks: "How should this system think, interpret, and intervene responsibly?"

01 · Interpretation
Frame friction correctly
Blockers are executive-function difficulty — never laziness, poor character, or moral failure.
02 · Epistemic
Know what the system knows
Distinguish between what is known, inferred, recommended, and uncertain. No overreach.
03 · Anti-shame
No punitive states
No streak-loss punishment, no red failure defaults, no silent conversion of missed actions into moral pressure.
04 · Human-centered
Respect cognitive limits
Sleep, recovery, attention bandwidth, and emotional context are first-class constraints — not obstacles to optimize around.
05 · Confidence
Calibrated recommendations
The system knows when to recommend strongly, when to offer choices, and when to say "not enough context."
06 · Practicality
Observable and testable
All prescriptions must be concrete enough to validate in real life. No interventions that can't be verified.

Closed-Loop Architecture

The system that teaches itself.

Unlike static productivity tools, the ADHD O/S improves its recommendations over time — within guardrails. The Governance Layer sets the rules for interpretation and intervention. The Adaptation Layer generates updated recommendations. The Intelligence Layer becomes sharper. The loop tightens, responsibly.
